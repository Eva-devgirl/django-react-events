# Django + React Events App
A simple full-stack Events application built with **DJango REST Framework** ans **React (Vite)**.
Users can view events fetched from a Django API.

---

- Django REST Framework (backend)
- React + Vite (frontend)

## Features
- View events list
- Django REST API
- React frontend fetching data from API
- CORS configured for frontend-backend communication

---

## Tech Stack

### Backend:
- Django
- Django REST Framework
- Django-cors-headers

### Frontend:
- React
- Vite
- Axios / Fetch API
